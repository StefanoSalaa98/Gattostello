# 🐱 Gattostello

Web app per la gestione e adozione dei gatti di un rifugio.

## 🚀 Tecnologie
Frontend:
- React
- Bootstrap

Backend:
- Node.js
- Express
- MySQL

## ⚙️ Installazione

Clonare il repository

git clone ...

### Backend

cd backend
npm install
npm run start
npm run watch

### Frontend

cd frontend
npm install
npm run dev
